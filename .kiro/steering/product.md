# Bluesky Social Product Overview

## What is Bluesky Social?

Bluesky Social is a cross-platform social media application (v1.113.0) that connects users to the AT Protocol (atproto) decentralized social network. It provides a modern, accessible social experience across iOS, Android, and Web platforms.

## Purpose

Bluesky enables users to participate in a decentralized social network built on open protocols, offering an alternative to traditional centralized social platforms.

## Key Features

- **Cross-platform support**: Native iOS and Android apps via React Native, plus a web version
- **Decentralized architecture**: Built on AT Protocol for user data ownership and portability
- **Real-time interactions**: Posts, replies, likes, reposts, and direct messaging
- **User profiles**: Customizable profiles with avatars, bios, and follower management
- **Feed algorithms**: Personalized and algorithmic feed options
- **Search and discovery**: Find users, posts, and trending content
- **Internationalization**: Multi-language support via Lingui
- **Accessibility**: Built with accessibility standards in mind
- **Notifications**: Real-time notifications for interactions
- **Moderation tools**: User controls for blocking, muting, and reporting

## Platform Details

- **iOS**: Native app via React Native and Expo
- **Android**: Native app via React Native and Expo
- **Web**: Full-featured web application served by Go backend (bskyweb)
